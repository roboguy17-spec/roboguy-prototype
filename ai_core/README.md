Robo Guy is an AI companion designed to support emotional well-being, personal growth, and discipline through emotionally intelligent, adaptive, and ethical reasoning.

This repository contains the early-stage prototype and design logic for Robo Guy, developed for the Hult Prize competition.

## Core Features
- Emotion detection from speech-to-text
- Adaptive cloud-based memory
- Divine Intelligence (ethical, faith-aware reasoning)
- Voice-based interaction
- Gen-Z styled animated companion (UI handled separately)

## Project Status
This is an early prototype focused on validating emotional intelligence, adaptation logic, and user interaction flow.
